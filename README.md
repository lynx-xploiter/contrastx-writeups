# contrastx| Cybersecurity Writeups 🚩

A collection of detailed writeups covering CTF challenges, VAPT research, and technical walkthroughs. Hosted at [contrastx-writeups.vercel.app](https://contrastx-writeups.vercel.app/).

## 📖 Content Overview

* **CTF Walkthroughs:** Solutions and methodologies for various platforms (HackTheBox, TryHackMe, etc.).
* **VAPT Research:** Documentation of vulnerability assessments and penetration testing findings.
* **Technical Notes:** Deep dives into specific exploits, tools, and security concepts.

## 🛠️ Built With

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Dynamic Content:** Custom JavaScript-based post rendering from `db.js`.
* **Deployment:** Vercel (CI/CD)
* **SEO:** Optimized metadata for technical content sharing.

## 🚀 Navigation

The writeups are categorized and searchable through the web interface to make finding specific vulnerabilities or platforms easier.
